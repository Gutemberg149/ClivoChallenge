# 🐾 MedVet - Gestão Veterinária Inteligente

O **MedVet** é uma solução digital completa para o gerenciamento da saúde e bem-estar animal. O projeto foi concebido para centralizar informações vitais de pets, permitindo que tutores e profissionais veterinários acompanhem históricos médicos, rotinas de cuidados e tratamentos terapêuticos para que tenha tratamento preventivo se caso for necessario.

---

## 💡 A Solução

O aplicativo resolve o problema da separação de informações sobre a saúde animal, oferecendo módulos específicos para:

- **🛡️ Módulo Preventivo:** Registro detalhado de vacinas, vermífugos e exames com histórico completo.
- **💊 Módulo Terapêutico:** Controle de medicamentos, dosagens e horários de administração.
- **☀️ Módulo Bem-Estar:** Monitoramento de atividades físicas, alimentação e observações comportamentais.
- **🎨 Experiência Adaptativa:** Suporte nativo ao **Dark Mode**, garantindo conforto visual em qualquer ambiente.
- **📂 Organização por Espécies:** Cadastro segmentado para diferentes tipos de animais.

---

## 🛠️ Tecnologias e Arquitetura

O ecossistema MedVet utiliza tecnologias de ponta para garantir performance e escalabilidade:

### **Mobile (Frontend)**

- **React Native (Expo):** Framework para desenvolvimento mobile cross-platform.
- **Context API:** Gerenciamento de tema dinâmico (Light/Dark Mode).
- **React Navigation:** Sistema de rotas e navegação fluida.
- **AsyncStorage:** Armazenamento local persistente para dados do usuário.
- **Vector Icons:** Interface rica com ícones intuitivos.
